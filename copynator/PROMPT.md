Create a chrome/edge extension to meet the following goal: extract content from a website, map it, and paste it into a second website. The goal is to make it easy to transfer informations from one website to another.

## Directives

- The extension should be flexible, allowing the user to map any website forms, fields and main informations, letting they ready to be pasted into another website. The extension also should allow mapping of the data obtained from the first website into the fields of the second.
- The extension should be able to save the mappings the user makes (as an option).
- The extension should work locally, not depending on the internet.
- Make the extension simple to use and intuitive. 
- Keep in mind that the goal is to improve productivity in these data from multiple website scenarios.
