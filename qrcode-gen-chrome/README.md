# QR Rápido

Extensão leve para Chrome e Microsoft Edge que gera QR Codes a partir de texto colado ou digitado no popup. Tudo roda localmente, sem backend, sem rede e sem histórico.

## Como instalar em modo desenvolvedor

1. Abra `chrome://extensions` no Chrome ou `edge://extensions` no Edge.
2. Ative o modo de desenvolvedor.
3. Escolha carregar extensão sem compactação.
4. Selecione esta pasta: `C:\PROJETOS\personal\qrcode-gen-chrome`.

## Como usar

1. Clique no ícone da extensão.
2. Cole ou digite o texto.
3. Escaneie o QR Code, copie a imagem ou baixe o PNG.

## Privacidade

A extensão não salva histórico, não lê páginas abertas e não usa internet. A única permissão solicitada é `clipboardWrite`, usada para copiar o QR Code como imagem.
