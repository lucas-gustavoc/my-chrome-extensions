# Chrome Extensions

This repository collects personal browser extension projects, mostly focused on small, local-first tools for Chrome-compatible browsers.

## Projects

| Project | Description |
| --- | --- |
| [qrcode-gen-chrome](./qrcode-gen-chrome) | Lightweight Chrome and Microsoft Edge extension that generates QR Codes from pasted or typed text in the popup. It runs locally, without a backend, network access, or history tracking. |

## Working With This Repository

Each extension lives in its own folder and should include its own README with installation and usage notes.
